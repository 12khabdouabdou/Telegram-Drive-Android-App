# Codemap: Mobile Components

**Responsibility**: Contains UI components tailored for mobile devices (e.g., touch-friendly lists, bottom nav bars, action popovers, `MobileDashboard`).

**Design Patterns**: Touch-Optimized Views, Overlay/Popover Patterns, Responsive Mobile Layouts.

**Data & Control Flow**: Data flows from global state or contexts down to mobile-specific presentation layers. Touch actions trigger state updates or navigation.

**Integration Points**: Consumed by the main App router for mobile environments. Uses shared utilities and types.
